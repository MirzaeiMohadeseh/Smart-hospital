# Smart Hospital (Java Console Project)

## Project Description

Smart Hospital is a **simple console-based Java application** developed to practice **basic Object-Oriented Programming (OOP) concepts**. The project simulates a small hospital system using text-based menus and console outputs.

---

## Key Concepts Used

* Object-Oriented Programming (OOP)
* Encapsulation
* Inheritance
* Abstraction
* Interface and Polymorphism

---

## Project Type

* Console Application (CLI)
* No graphical user interface (GUI)
* User interaction through console input/output

---

## Main Features

* Register patients
* Doctor appointment and basic diagnosis
* Drug prescription
* Nursing services
* Pharmacy drug delivery

---

## How to Run

1. Compile the project:

   ```bash
   javac hospital/Main.java
   ```
2. Run the program:

   ```bash
   java hospital.Main
   ```

---

## Notes

This project is **educational and beginner-level**, designed to demonstrate Java OOP fundamentals in a practical example.
